Simple Gid+

Simple Gid+ provides you with a lightweight (~2KB uncompressed) and responsive CSS grid & utility components for easier front-end development.

  • Tiny in size (~2KB uncompressed).
  • Easy to use.
  • lightweight, widely compatible.
  • Supports nesting.
  • Bootstrap-like in syntax.

Creating a Grid

Simple Grid+ uses a 12-column CSS grid that's easy to use. Simply create a container, add a row, and create all of your columns! 

      
        <div class="container">
          <div class="row">
            <div class="col-3">
              <!-- This content will take up 3/12 (or 1/4) of the container -->
            </div>
            <div class="col-3">
              <!-- This content will take up 3/12 (or 1/4) of the container -->
            </div>
            <div class="col-6">
              <!-- This content will take up 6/12 (or 1/2) of the container -->
            </div>
          </div>
        </div>

Grid Nesting

Simple Grid+ fully supports nesting grids as seen above and below.

      
        <div class="container">
          <div class="row">
            <div class="col-3">
              <div class="row">
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
              </div>
            </div>
            <div class="col-3">
              <div class="row">
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
              </div>
            </div>
            <div class="col-6">
              <div class="row">
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
                <div class="col-6">
                  <!-- This content will take up 6/12 (or 1/2) of the container -->
                </div>
              </div>
            </div>
